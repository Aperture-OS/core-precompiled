# Core-repo-precompiled
Core repository of ApertureOS for Blink-PM, precompiled packages only.

> [!IMPORTANT]\
> NOT PRODUCTION READY!


## Contribute

Guidelines: 
* Compile the packages into a folder, and tar the folder ( possibly as .tar.gz ), then place the archive in `/packages/name-version.extension`
* Make a Pull request in order to do this
* If you want to you can test the package on a virtual machine (make a minimal rootfs if you know how to, i'll release a basic testing iso when i can), if you do test the package, please send a screenshot of it working, thank you!

You will be given the Contributor and Repository Maintainer role on our Discord Server, your name will also get printed on this repository footer. Thank you for contributing!!!

For other informations please follow the instructions in [CONTRIBUTING.md](https://github.com/Aperture-OS/core-precompiled/blob/main/CONTRIBUTING.md) for guidelines on contributing to Core Repository Precompiled.
